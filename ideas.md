# Elastic Trend Live Scanner — Landing Page Design Ideas

## Response 1 — Terminal Noir
<response>
<text>
**Design Movement:** Cyberpunk Terminal / Dark Data Visualization

**Core Principles:**
- Raw, monospaced data aesthetic — everything looks like it came out of a trading terminal
- High contrast: near-black backgrounds with electric green and amber accents
- Information density — the page itself feels like a live data feed
- Asymmetric layouts with hard grid lines and ruled separators

**Color Philosophy:**
- Background: #080c10 (near-black with a blue-black tint)
- Primary accent: #00ff88 (electric terminal green)
- Secondary accent: #f59e0b (amber/gold for alerts)
- Bear accent: #ff3b5c (red-pink)
- Text: #c8d6e5 (cool off-white)
- Muted: #4a5568
- Emotional intent: urgency, precision, professional edge

**Layout Paradigm:**
- Full-bleed hero with a live-looking ticker tape scrolling at the top
- Two-column asymmetric layout: 60% content / 40% mock terminal output
- Horizontal rule separators between sections with section labels in monospace
- Feature cards styled as terminal windows with fake `$` prompt headers

**Signature Elements:**
- Blinking cursor `_` in headlines
- Fake live scan output panel in the hero (animated rows appearing one by one)
- Section dividers styled as `# ============ SECTION NAME ============ #`

**Interaction Philosophy:**
- Hover states reveal more data (like expanding a row in a scanner)
- Buttons styled as CLI commands: `[ RUN SCAN ]`
- Scroll-triggered line-by-line text reveals

**Animation:**
- Ticker tape scrolling horizontally at the top
- Hero scan panel: rows fade in one by one, simulating a live scan running
- Number counters animate up on scroll-into-view
- Subtle scanline overlay on the hero background

**Typography System:**
- Headlines: JetBrains Mono Bold (monospace, technical authority)
- Body: IBM Plex Sans (readable, technical but not sterile)
- Data labels: JetBrains Mono Regular
</text>

- Features in a 3-column grid with icon-topped cards

**Signature Elements:**

**Typography System:**
- Headlines: Syne ExtraBold (geometric, modern authority)
- Body: DM Sans (clean, highly readable)

## Response 3 — Elastic Signal
<response>
<text>
**Design Movement:** Dark Brutalism meets Quantitative Finance — raw structure, bold type, signal-forward

**Core Principles:**
- Bold typographic hierarchy — headlines are massive, unapologetic
- Structural honesty: grid lines, borders, and dividers are visible design elements
- Signal over noise: every section answers one question clearly
- Tactile depth: subtle noise texture on backgrounds, not flat

**Color Philosophy:**
- Background: #0a0e14 (charcoal-black)
- Surface: #111820 (slightly lighter for cards)
- Primary: #00d4aa (teal-green — the "elastic" color)
- Bull: #22c55e (green)
- Bear: #ef4444 (red)
- Accent: #3b82f6 (blue for secondary elements)
- Text: #f1f5f9 (near-white)
- Muted: #64748b
- Emotional intent: confidence, momentum, clarity

**Layout Paradigm:**
- Full-width hero with a large left-aligned headline and a right-side mock dashboard panel
- "How It Works" as a horizontal step-flow (not vertical)
- Features in a 2-column layout with large numbered headings
- Pricing as a single featured card (one tier) with a bold CTA

**Signature Elements:**
- Large bold section numbers (01, 02, 03) as decorative anchors
- Teal horizontal rule lines as section separators
- Mock scan result table with live-looking data in the hero

**Interaction Philosophy:**
- Hover on feature items reveals a teal left-border accent
- Scroll-triggered entrance animations (slide up + fade)
- CTA button has a shimmer sweep animation on hover

**Animation:**
- Hero: mock scan table rows appear one by one on load
- Section numbers count up on scroll
- Feature items stagger in from left
- Gradient background subtly shifts hue over time

**Typography System:**
- Headlines: Space Grotesk Bold (geometric, distinctive, not Inter)
- Body: Inter (readable, familiar — used only for body, not headlines)
- Monospace: JetBrains Mono (for scanner data mockups)
</text>

## Selected Design: Response 3 — Elastic Signal

**Rationale:** The "Elastic Signal" approach directly mirrors the product's identity — teal-green accents match the scanner's color scheme, the mock dashboard in the hero immediately communicates what the product does, and the bold typographic structure gives it a premium, confident feel without being overly stylized. The large section numbers and horizontal step-flow for "How It Works" make the page scannable and clear for traders who value efficiency.
